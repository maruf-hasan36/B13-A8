📚 GoodReads Web App

A modern and fully responsive Book Management Web Application built with Next.js, React, and Tailwind CSS. This platform allows users to explore books, read and share reviews, and manage their personal profiles with a smooth and secure experience.

🎯 Project Overview

This project is designed to create a simple yet powerful platform where users can:

Discover and explore books
View detailed book information
Share their thoughts through reviews
Manage personal profiles securely

🚀 Features
🔐 Secure Authentication
User login and registration system with email/password and Google sign-in.
📖 Book Browsing
Explore books with search and category filtering.
🔒 Protected Routes
Certain pages like book details and user profile are only accessible after login.
⭐ Review System
Users can add and view reviews for different books.
🎞️ Smooth Animations
Interactive UI with clean transitions for better user experience.
📢 Dynamic Marquee
Displays announcements like new arrivals and offers.
👤 User Profile Management
Update personal information including name and profile image.
🔔 Notifications
Real-time feedback for user actions like login, update, etc.
🛠️ Tech Stack
Next.js – For routing and server-side rendering
React – For building UI components
Tailwind CSS – For styling and responsive design
Framer Motion – For animations
Swiper.js – For interactive sliders
React Toastify – For notifications
BetterAuth – For authentication system
